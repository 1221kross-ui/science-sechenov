Институт бионических технологий и инжиниринга

Миссия института – транслировать принципы функционирования живых систем в инженерные решения для биомедицины посредством трансдисциплинарного подхода.

Разработка сенсоров для медицины
Создание имплантируемых нейростимулирующих протезов
Разработка лазера для сварки биологических тканей

Структура института

Лаборатория биомедицинских нанотехнологий

Лаборатория биомедицинских нанотехнологий

  1. Исследование биологических свойств нанокомпозитных покрытий для имплантатов связок суставов
  2. Исследование биологических свойств имплантационного наноматериала для восстановления костно-хрящевых дефектов
  3. Исследование метода лазерного сваривания биотканей
  4. Исследование тканеинженерных конструкций для восстановления дефектов тканей сердечно-сосудистой системы
Лаборатория носимых биосовместимых устройств и бионических протезов

Лаборатория носимых биосовместимых устройств и бионических протезов

  1. Разработка персонализированных диагностических систем призванных обеспечить помощь в принятии решений при ранней диагностике социально значимых заболеваний.
  2. Разработка искусственных органов и бионических протезов.
  3. Разработка гемосовместимых антикоагулянтных покрытий имплантов с целью повышения антитромбогенной активности в области прямого контакта с кровью.
  4. Разработка антимикробных покрытий для предотвращения риска инфекции во время имплантации искусственных протезов и органов.

Имплантируемые нейростимулирующие протезы

Устройства для стимуляции головного и спинного мозга, органов желудочно-кишечного тракта.

Совместный проект с АО “ЗИТЦ” и МИЭТ.

Системы для стимуляции коры головного мозга и глубокой стимуляции мозга

Применимы при: болезнь Паркинсона, эпилепсия, эссенциальный тремор, дистония, обсессивно- компульсивное расстройство, головные боли, депрессия, инсульт, нейропатическая спондилопатия.

Имплантируемые системы для стимуляции спинного мозга и периферических нервов

Применимы при: дорсалгия, боль в конечности, спастичность, нервно-мышечная дисфункция мочевого пузыря, дисфагия.
